# 產品需求文件：股票助手 LINE Bot

## 1. 產品概述

建立一個 LINE Bot「股票小助手」，讓使用者透過 LINE 訊息即時查詢台灣股票價格、
獲得 Gemini AI 的股票分析建議，並管理個人自選股追蹤清單。

本專案以「可以執行、能回覆訊息」為核心目標，提供簡潔實用的股票查詢體驗。

## 2. 目標用戶

| 用戶類型 | 特徵 | 需求 |
|----------|------|------|
| 散戶投資人 | 每日關注股市動態 | 快速查詢即時股價 |
| 上班族 | 工作中無法開看盤軟體 | 用 LINE 即時查看股價 |
| 股市初學者 | 對技術分析不熟 | AI 分析建議作為參考 |

## 3. 核心功能

### 3.1 即時股價查詢
- **觸發方式**：傳送 `查詢 <股票代號或名稱>`
- **資料來源**：twstock（台灣證券交易所）
- **回覆內容**：股票名稱、即時價格、漲跌、漲跌幅、成交量
- **錯誤處理**：查無股票時回覆友善提示

### 3.2 AI 股票分析
- **觸發方式**：傳送 `分析 <股票代號或名稱>`
- **AI 引擎**：Google Gemini 2.0 Flash
- **分析內容**：技術面分析、基本面概述、投資建議
- **回覆限制**：500 字以內的繁體中文分析

### 3.3 追蹤清單管理
- **新增追蹤**：傳送 `追蹤 <股票代號>`
- **取消追蹤**：傳送 `取消追蹤 <股票代號>`
- **查看清單**：傳送 `我的清單`
- **儲存方式**：SQLite 資料庫，按使用者 ID 分別儲存

### 3.4 使用說明
- **觸發方式**：傳送 `幫助` 或 `help`
- **回覆內容**：所有可用指令的說明

## 4. 非功能需求

- **回應速度**：股價查詢 < 3 秒，AI 分析 < 10 秒
- **可靠性**：API 呼叫失敗時有 fallback 錯誤訊息
- **安全性**：所有 token 存放在 `.env`，不進入版本控制
- **可維護性**：程式碼加上中文註解和 docstring

## 5. 技術架構

```
LINE App → LINE Platform → ngrok → FastAPI (app.py)
                                        ├── twstock（查股價）
                                        ├── Gemini API（AI 分析）
                                        └── SQLite（追蹤清單）
```

| 元件 | 技術 |
|------|------|
| Web 框架 | FastAPI + uvicorn |
| LINE SDK | line-bot-sdk v3 |
| AI 引擎 | Google Gemini 2.0 Flash |
| 股票資料 | twstock |
| 資料庫 | SQLite3 |
| 隧道 | ngrok |

## 6. 資料庫設計

### watchlist（追蹤清單）
| 欄位 | 型別 | 說明 |
|------|------|------|
| id | INTEGER PK | 自增主鍵 |
| user_id | TEXT | LINE 使用者 ID |
| stock_id | TEXT | 股票代號 |
| stock_name | TEXT | 股票名稱 |
| added_at | TIMESTAMP | 新增時間 |

### query_log（查詢紀錄）
| 欄位 | 型別 | 說明 |
|------|------|------|
| id | INTEGER PK | 自增主鍵 |
| user_id | TEXT | LINE 使用者 ID |
| stock_id | TEXT | 股票代號 |
| stock_name | TEXT | 股票名稱 |
| query_type | TEXT | 查詢類型 (price/analysis) |
| queried_at | TIMESTAMP | 查詢時間 |
